# 🎬 提出用デモ動画の作成

## Summary
ハッカソン提出用のデモ動画を作成し、YouTubeにアップロードする。
この動画はZenn記事やREADMEに埋め込み、審査員にアプリの動作と魅力を短時間で伝えるために使用する。

## Tasks
- [ ] デモシナリオの策定（「ぼくの夏休み」風の日記生成フローを見せる）
- [ ] 画面収録（QuickTime Player等を使用、音声入力の様子も含める）
- [ ] 動画編集（不要な部分のカット、字幕・注釈の追加）
- [ ] YouTubeへのアップロード（限定公開設定など）
- [ ] 動画URLの取得

## Notes
- 審査員は数多くの作品を見るため、冒頭30秒で魅力を伝える工夫が必要。
- 音声対話の様子がわかるように、システム音声とマイク入力の両方を収録する必要がある。
- 動画の長さは1分〜3分程度が目安（要確認）。
